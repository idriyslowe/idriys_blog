Ruby Practice Round 2

Determine if two strings are anagrams (case insensitive)

Driver code:

            p anagram.anagrams?("cinema", "iceman")

            p anagram.anagrams?("Cinema", "IceMan")

            p anagram.anagrams?("baseball", "ballbass")

My solution:

            class Anagrams

              def anagrams?(word1, word2)

                array1 = word1.downcase.split('').sort

                array2 = word2.downcase.split('').sort

                array1 == array2

              end

            end

            anagram = Anagrams.new

The Anagrams class I wrote only has one method called "anagrams?". It takes in two words as arguments ("word1" & "word2"). Each word is made completely lowercase, turned into an array of letters, and sorted. Downcasing ensures the letters all sort equally. Now "word1"'s sorted letters are stored in "array1" while "word2" is stored in "array2".

The last line of the "anagrams" method checks the equality of the two arrays and will spit out "true" or "false". This is the friendly reason the "anagrams?" method ends with a question mark. Oftentimes, that operator is a choice the writer made, or a hint that the method results in a boolean. It was my first hint when deciphering the driver code.
