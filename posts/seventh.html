Bootcamp Final Project

Part 2

Studere was an undertaking.

This was an app I was building secretly for myself. It would allow me to collect thoughts (in writing), screenshots, and links for any subject one desired to study. This summarizes the MVP (minimum viable product) anyway but I want it to do so much more.

Here's a slightly convoluted breakdown/story:

A user logs in and sees a list of all of their studying "Environments". An environment can be a broad subject like programming (duh). Once a user clicks on the environment they wish to work in, the subject breaks down to "Note Pages". A note page should be more specific like Javascript. Once a user clicks on a note page to work on, they see all the notes which are meant to be free association like a sticky would be. Finally, once the user has found a useful link on the interwebs, they can screencap the page using the Studere extension and the image along with url will save to the user's notepage to be (drag and drop) associated with an appropriate note.

The ultimate goal is for this app to gather a community of studiers so Studere would give users the ability to comment on one anothers' "Note Pages". Users would be able to toggle the privacy of each environment... you know... in case you didn't want anyone knowing you were studying to rob a bank.

This ultimate goal includes being able to create calendar events/reminders and even receive SMS message reminders.
