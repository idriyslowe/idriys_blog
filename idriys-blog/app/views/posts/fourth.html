Flatten this array:

This first problem made me nervous at first glance. Since I've been learning Rails and Bootstrap, I hadn't practiced writing any simple Ruby programs for weeks.

array = ["1", 3, "Home", [4], "Word", [3, "Eight"]]

flat_array = []
array.each do |item|
if item.is_a? Array
item.each do |item|
flat_array << item
end
else flat_array << item
end
end
p flat_array

First, I knew I needed to create an array ("flat_array") to store the new flattened array.

If the array was filled with non-array items, flattening wouldn't be neccessary. I needed a way to deal with the nested arrays. Simple enough... find the arrays with an if statement so I could treat them differently during the rest of the block.

If the item in the array is another array, iterate through each item in the inner array and shovel them into the array "flat_array".

For all other items (which are luckily not collections in this example) simply shovel with no fanfare.

Et Voil√•! I get to print my new flattened array which remains in the same order as the original arrays.

Calculate the average (mean) of an array of numbers:

Driver code:
p average([3, 2, 1]) should be 2
p average([8, 9]) should be 8.5
p average([1, 1, 1, 2]) should be 1.25

input_array = [8, 9]
total = 0.0
input_array.each do |num|
total += num
end
average = total / input_array.length
p average

This probably isn't the cleanest code you've ever seen for this type of problem but I wrote it in less than 15 minutes so give me some credit. Early in this program, I need to create a variable that will hold the total of all numbers and since I know the average won't neccessarily be a whole number, I'm declaring the value a float by assigning the decimal, 0.0.

I iterate through the array, adding each number to the total using the += abbreviated assignment operator. After all that hoopla, I create a variable that will hold the final answer which is the total divided by the original array's length.
